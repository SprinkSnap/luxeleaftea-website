# Luxe Leaf Tea — Logo & Branding Recommendations

## Should the logo include a tagline?

**Recommended: No tagline inside the logo.**

Use **icon + brand name only** in the primary logo lockup. Reasons:

- **Scalability** — Taglines disappear or clutter at favicon, mobile header, and packaging small print sizes.
- **Web conversion** — The header logo should be instantly readable; the value proposition belongs in the hero headline and CTA, not the mark.
- **Flexibility** — Taglines change with campaigns; the logo should not.

### Where the tagline should live instead

| Placement | Example |
|-----------|---------|
| Homepage hero (H1/subhead) | "Premium Chinese Loose-Leaf Tea, Delivered Fresh" |
| Meta description / SEO | Already in `seo-meta.html` |
| Packaging back panel | Origin story, brewing notes |
| Email subject lines | Campaign-specific |

**Optional secondary line** (only on packaging or wide hero layouts, never in favicon):

> Chinese Loose-Leaf Tea

Keep it to 3–4 words max if used at all.

## What the logo should communicate

The mark alone should signal:

1. **Tea** — Leaf motif (not a cup, teapot, or mug — those say "cafe," not premium loose leaf)
2. **Chinese origin** — Longjing-style needle leaves, subtle 茶葉 characters, ink-and-jade palette
3. **Premium** — Gold restraint, wide-tracked serif typography, generous negative space

## Current v2 design rationale

- **Crossing dual leaves** — Distinctive silhouette (not generic single clipart leaf); reads as fresh bud / mountain peak
- **茶葉 characters** — Clear Chinese tea heritage without cliché lanterns or dragons
- **No ornate borders** — Avoids South Asian/paisley associations; stays East Asian minimal
- **Name only** — Clean for website header click-through to shop
- **Horizontal lockup** — Icon left, wordmark right; optimized for e-commerce navigation bars
