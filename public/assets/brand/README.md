## SEO & conversion integration

- **`seo-meta.html`** — favicon links, Open Graph, Twitter Card, JSON-LD Organization schema
- **`logo-head.html`** — semantic header logo with WebP + PNG `srcset`, alt text, and explicit dimensions
- **`structured-data.json`** — Organization schema for CMS or build pipelines
- **`/site.webmanifest`** — PWA manifest with brand icons
